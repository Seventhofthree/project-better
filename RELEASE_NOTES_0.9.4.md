# Pathfinder 0.9.4 Evening Review / Weekly Review

## Purpose

Make the Review page feel more like a daily companion and less like a raw report.

## What changed

- `APP_VERSION` is now `0.9.4`.
- Review page now has an **Evening review** card.
- Evening review shows:
  - today score
  - food loop
  - movement status
  - routine completion
  - what went well
  - what to adjust
  - next best step
- Added a **Weekly dashboard** card.
- Added weekly coaching cards:
  - what went well
  - what got in the way
  - next best focus
- Kept the copy-ready weekly review packet.
- Kept 0.9.3 progress trend polish.
- Kept 0.9.2 exercise/routine polish.
- Kept 0.9.1 meals polish.
- Kept 0.9.0.1 Settings and Update Safety fixes.
- No persistence behavior was changed.

## Why this matters

This is where Pathfinder starts feeling more like a companion: it closes the day, explains the week, and gives one practical next action.

## Done When

- Existing saved data survives the update.
- Review page opens.
- Evening review card appears.
- Weekly dashboard card appears.
- Weekly coaching cards appear.
- Copy review buttons still work.
- Settings still opens.
- Run save test still passes.
- Refresh still preserves data.
