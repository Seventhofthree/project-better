# Pathfinder 0.8.8 Update Package

Copy these files into the repo root and overwrite existing files:

- `index.html`
- `persistence-bootstrap.js`
- `service-worker.js`
- `RELEASE_NOTES_0.8.8.md`
- `docs/ROADMAP.md`

Keep existing files:

- `app.js`
- `styles.css`
- `manifest.webmanifest`
- `icon.svg`
- `README.md`
- `ROADMAP.md` if you still have it in the root

After upload:

1. Open Pathfinder.
2. Refresh once or twice.
3. Go to Settings.
4. Confirm the App version card says `0.8.8 Data Safety Cleanup`.
5. Confirm the Data safety card appears.
6. Run the Done When tests in `RELEASE_NOTES_0.8.8.md`.
