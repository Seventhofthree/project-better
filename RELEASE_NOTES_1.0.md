# Pathfinder 1.0 Stable

## Purpose

Promote the fully passed 0.9.9 Release Candidate to the first stable Pathfinder release.

## Included

- Today dashboard and companion guidance
- Meal tracking, calorie estimates, repeat foods, and saved foods
- Exercise plans, minimum wins, recovery choices, and movement guidance
- Routine tracking
- Morning brief, next-action stack, and evening wind-down coaching
- Progress and weight-trend context
- Evening and weekly review
- History and data export
- Mobile/PWA polish
- Local-first saving with backup and recovery support
- Settings save test and release-readiness checks
- Safer render fallback

## Stability statement

- Built directly from the passed 0.9.9 Release Candidate.
- No persistence behavior was changed.
- No major feature was added.
- The Settings hotfix and render fallback remain in place.
