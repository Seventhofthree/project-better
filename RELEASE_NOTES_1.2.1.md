# Pathfinder 1.2.1 — Navigation Hierarchy Polish

## Purpose

Clarify that Today, Food, Movement, and Progress are everyday destinations while Settings is an occasional utility.

## Changes

- Moved Settings out of the daily navigation row.
- Added a standalone circular Settings gear in the upper-right.
- Moved the date controls beneath the page title and aligned them left.
- Kept the four daily destinations together beneath the date controls on desktop.
- Kept only the four daily destinations in the phone bottom navigation.
- Preserved all nested views, internal jump buttons, IndexedDB data, snapshots, backups, imports, and exports.

## Data impact

None. Pathfinder 1.2.1 uses the same schema and storage foundation as 1.1 and 1.2.
