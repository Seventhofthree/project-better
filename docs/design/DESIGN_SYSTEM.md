# Pathfinder Design System

## Design Personality

Calm. Clean. Intentional. Premium. Minimal.

Pathfinder should feel like a high-quality tool, not a busy productivity app.

## Core Principle

The app should always help answer:

> What is the next best thing I can do?

## Colors

Initial palette:

- Navy: navigation / primary actions
- Forest Green: health / success
- Orange: nutrition
- Purple: progress
- Charcoal: text
- Light Gray: backgrounds and borders
- White: cards and content surfaces

## Component Principles

- Large touch targets
- Minimal text
- Clear hierarchy
- One primary action per screen
- Daily actions reachable in one tap

## Core Components

### Metric Card

Used for weight, sleep, streaks, water, brush head age, and milestones.

### Checklist Card

Used for routines, workouts, meal prep, and weekly resets.

### Progress Card

Used for weekly completion and habit streaks.

### Reminder Card

Used for brush head replacement, retinol nights, meal prep, appointments, and upcoming tasks.

### Quick Action Button

Used for starting high-frequency actions:
- Start Morning Routine
- Start Workout
- Log Weight
- View Meals

## Voice and Tone

Calm and direct.

Examples:
- Morning complete. Have a good day.
- Workout complete. Nice work.
- Retinol tonight.
- Brush head replacement due soon.

Avoid:
- Guilt
- Over-cheering
- Nagging
- Excessive notifications
