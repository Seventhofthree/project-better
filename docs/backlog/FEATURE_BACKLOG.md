# Feature Backlog

## Must Have — Version 1.0

### Home Dashboard
Shows today's focus, routine status, workout, meals, weight, next milestone, and reminders.

### Morning Routine
Guided routine using one-step-at-a-time completion.

### Night Routine
Guided routine with retinol schedule support.

### Today's Workout
Displays the current workout with simple completion tracking.

### Today's Meals
Displays the current meal plan and basic prep guidance.

### Weight Logging
Fast daily/weekly weight entry.

### Progress Overview
Shows basic trend, milestones, and completed habits.

## Should Have — Version 1.1

### Shopping List
Generates from meal plan and personal care refill needs.

### Meal Prep Reminder
Prompts before planned prep days.

### Brush Head Tracker
Tracks Sonicare brush head age.

### Retinol Schedule
Shows which nights are retinol nights.

### Monthly Review
Simple reflection and progress check.
