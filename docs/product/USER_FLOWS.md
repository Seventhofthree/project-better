# User Flows

## Open App in the Morning

1. Open Pathfinder.
2. Land on Home.
3. See today's date and focus.
4. Tap Start Morning Routine.
5. Complete routine one step at a time.
6. Return to Home.
7. Home shows Morning Routine complete.

## Log Weight

1. Open Pathfinder.
2. Tap Log Weight from Home.
3. Enter weight.
4. Save.
5. Home updates current weight.
6. Progress updates trend and milestone distance.

## Complete Workout

1. Open Pathfinder.
2. Tap Today's Workout.
3. Complete workout steps.
4. Tap Finish Workout.
5. Home updates workout status.
6. Progress updates weekly completion.

## View Today's Meals

1. Open Pathfinder.
2. Tap Today's Meals.
3. See Meal 1 and Meal 2.
4. Tap a meal to view ingredients and prep notes.
5. Return to Home.

## Night Routine

1. Open Pathfinder in evening.
2. Tap Start Night Routine.
3. Complete routine.
4. Retinol step appears only on scheduled nights.
5. Return to Home.
