# Project Better OS — Product Requirements Document

## App Name

**Pathfinder**

## Vision

Pathfinder is a personal operating system that helps Joshua live intentionally by organizing daily routines, health, nutrition, fitness, goals, and life information into one trusted place.

Its purpose is not to collect data.

Its purpose is to reduce mental load.

## Mission

Build a single application that tells Joshua what he needs to do today, tracks what matters, and removes unnecessary decisions from daily life.

## Target User

Version 1.0 is built for one person:

**Joshua Kelly**

This is intentional. We optimize for real use first, then generalize later only if it makes sense.

## What Pathfinder Is

- Personal operating system
- Digital binder
- Daily dashboard
- Health hub
- Routine manager
- Long-term life companion

## What Pathfinder Is Not

- Social media
- Generic habit tracker
- Giant note-taking app
- Calendar replacement
- Budgeting app
- Another to-do list

## Version 1.0 Goal

Answer one question exceptionally well:

> What do I need to do today?

## Success Criteria

- Joshua opens it most mornings.
- Daily routines require less thinking.
- Important maintenance items are not forgotten.
- Progress is visible without duplicate data entry.
- The app feels calm, clean, and useful.
