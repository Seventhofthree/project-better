# Pathfinder 0.9.3 Progress Trend Polish

## Purpose

Make the Progress page easier to understand without changing the save system.

## What changed

- `APP_VERSION` is now `0.9.3`.
- Progress page now has a **Progress direction** dashboard.
- Added clearer 7-day scale-change wording.
- Added comparison against the prior 7-day window.
- Added a stronger **What this means** card.
- Added a **Scale context** card for water weight, sodium, soreness, sleep, stress, and logging gaps.
- Polished projection wording so it is clearly “paper math,” not a promise.
- Added a behavior trend card for minimum wins, full workouts, and recovery days.
- Kept 0.9.2 exercise/routine polish.
- Kept 0.9.1 meals polish.
- Kept 0.9.0.1 Settings and Update Safety fixes.
- No persistence behavior was changed.

## Why this matters

Weight changes are noisy. This release helps Pathfinder talk about trend, context, and next action instead of making the scale feel like a judgment.

## Done When

- Existing saved data survives the update.
- Progress page opens.
- Progress direction card appears.
- Weight trend chart still draws.
- Habit chart still draws.
- What this means card appears.
- Scale context card appears.
- Settings still opens.
- Run save test still passes.
